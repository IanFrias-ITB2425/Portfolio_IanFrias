Network Setup

In this project, I designed and implemented a secure network infrastructure for a small business. The goal was to create a reliable and efficient network that meets the client's needs while ensuring data security and network performance.

The project involved configuring routers, switches, and setting up VLANs to segregate network traffic. I also implemented firewall rules and VPN access for remote employees, ensuring secure connections to the company network.

Technologies Used:

  • Cisco Networking Equipment
  • Linux Servers
  • Network Design Tools
