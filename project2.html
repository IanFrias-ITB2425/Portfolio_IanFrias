Users & Passwords

In this project, I focused on user management and security within a virtual machine environment. I configured user accounts and passwords, and implemented security policies to enhance confidentiality and integrity.

The project involved setting up strong password policies, managing user permissions, and ensuring that the system adheres to best security practices. This experience deepened my understanding of system administration and cybersecurity principles.

Technologies Used:

  • Linux
  • Bash Scripting
  • VMware
